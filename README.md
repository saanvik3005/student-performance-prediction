# Student Performance Prediction Using Machine Learning

## Overview
A complete Machine Learning pipeline to predict whether a student will Pass or Fail based on key academic and social factors.

## Dataset
Kaggle - Student Performance Factors (6607 records)

## Features Used
- Hours Studied
- Attendance
- Parental Involvement
- Access to Resources
- Extracurricular Activities
- Exam Score

## Models Used
- Logistic Regression
- KNN
- SVM
- Naive Bayes
- Decision Tree
- Random Forest
- PCA + Random Forest

## Results
| Model | Accuracy |
|---|---|
| Logistic Regression | 100.00% |
| KNN | 97.35% |
| SVM | 99.92% |
| Naive Bayes | 95.31% |
| Decision Tree | 100.00% |
| Random Forest | 100.00% |
| PCA + Random Forest | 94.33% |

## Tools and Technologies
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
